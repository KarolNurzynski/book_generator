Prerequisites:
-- Python 3
    - To install python3 for Windows: https://www.python.org/downloads/windows/
    - To install python3 for MacOS: https://www.python.org/downloads/mac-osx/
    - To install python3 for other platforms: https://www.python.org/download/other/

-- virtualenv
    - To install virtualenv for windows you should have pip installed first(already installed with python).
    - pip install virtualenv
    - if you got an error update pip
        - to update pip in windows:
            - open power shell and run --> pip install --upgrade pip
        - to update pip on othe operating systems:
            - sudo pip install --upgrade pip

-- run the application unsing the run files
    - for windows double click on run.bat file
    - for other operating systems from the terminal run: service run.sh
